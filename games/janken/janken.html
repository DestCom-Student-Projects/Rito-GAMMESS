Découvre Janken, un dérivé de ce jeu avec des symboles peu communs !

SOUND ON

RULES

Les joueurs dévoilent simultanément leur "arme" symbolisée ci-dessous (diamant, papier toilette, couteau).
Le vainqueur est celui qui possède la meilleure combinaison.
Rappel des combinaisons :

  • Le diamant brise le couteau
  • Le papier toilette recouvre le diamant
  • Le couteau coupe le papier toilette

C'est toi qui lance la partie.
Pour cela, clique sur l'un des trois symboles pour réaliser une combinaison.
Bonne partie !
